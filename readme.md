### How to use

# List of tetrominoes
The program takes a file containing a list of tetrominoes as input.\n
Tetrominoes should be shown in this format, where # is a part of the piece, and . is an empty space

Example of a vertical line tetromino :
```
#...
#...
#...
#...
```

# Errors
The program will return an error if :
- a piece is composed of more or less than four squares
- the format is not as above for each pieace
- the file format is incorrect
